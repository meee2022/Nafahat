## 7) مسار الإصدار

1. **Testing → Closed testing** → أنشئ مسار → ارفع `app-release.aab`.
2. أضف **20 مختبِراً** (إيميلات Gmail) عبر قائمة بريد أو رابط.
3. اترك الاختبار **14 يوماً** متواصلة.
4. بعدها: **Production → Create release** → ارفع نفس الـ AAB (أو نسخة أحدث) → Submit.

> versionCode يُدار تلقائياً عبر EAS (`autoIncrement`). version الظاهر `1.0.0` في app.json.
